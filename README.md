# 我的世界玩家工具箱插件 (Minecraft-Player-Toolbox) - Minecraft Purpur 1.16.5

## 插件功能

这是一个为Minecraft Purpur 1.16.5版本设计的多功能玩家工具插件，支持中文语言，所有玩家（无OP或有OP权限）均可使用基本功能。

### 已实现的命令

1. **出生点传送**
   - `/spawn` - 返回服务器默认出生点（冷却时间：30秒）

2. **建议系统**
   - `/suggest [建议内容]` - 向管理员发送游戏建议，自动记录到文件

3. **家系统**
   - `/sethome` - 设置家的位置（冷却时间：15秒）
   - `/home` - 返回家的位置（冷却时间：20秒）
   - `/delhome` - 删除家的位置（冷却时间：10秒）

4. **传送请求**
   - `/tpa <玩家名>` - 向其他玩家发送传送请求（冷却时间：15秒）
   - `/tpaccept [玩家名]` - 接受传送请求
   - `/tpdeny [玩家名]` - 拒绝传送请求
   - `/tpcancel` - 取消发送的传送请求

5. **死亡返回**
   - `/back` - 返回死亡位置（冷却时间：20秒）

6. **皮肤管理**
   - `/skinreset` - 还原自己的皮肤（冷却时间：120秒）
   - `/skinreset <玩家名>` - 管理员重置其他玩家的皮肤

## 权限设置

- `playerutils.*` - 所有功能权限
- `playerutils.spawn` - 使用/spawn命令权限（默认所有玩家可用）
- `playerutils.suggest` - 使用/suggest命令权限（默认所有玩家可用）
- `playerutils.home` - 家系统相关命令权限（默认所有玩家可用）
- `playerutils.tpa` - 传送请求相关命令权限（默认所有玩家可用）
- `playerutils.back` - 使用/back命令权限（默认所有玩家可用）
- `playerutils.skinreset` - 皮肤还原命令权限（默认所有玩家可用）
- `playerutils.admin` - 管理员权限，可接收建议消息和重置其他玩家皮肤

## 使用说明

1. 将插件JAR文件放入服务器的`plugins`文件夹
2. 重启服务器以加载插件
3. 玩家可以直接使用所有基本命令，无需额外权限
4. 建议消息会保存在`plugins/PlayerUtils/suggestions.yml`文件中

## 注意事项

- 所有传送类命令都有冷却时间限制
- 传送请求有效期为30秒
- 家位置和死亡位置在服务器重启后会重置
- 皮肤还原功能会强制玩家重新连接以刷新皮肤
